# Canon Spec

---

## Repo structure

```
{repo}/
  canon/
    characters/{id}/definition.json
    worldbuilding/locations/{id}.json
  stories/{slug}/
    metadata.json          ← required for compliance
    chapter-01.md          ← episode content
    ko/chapter-01.md       ← optional locale chapters
    en/chapter-01.md
  .canonrc.json            ← author config
  canon.lock.json          ← generated by canon lock
  .canon-refs.json         ← generated by canon write
  CONVENTIONS.md
```

---

## metadata.json (v1.2)

```json
{
  "schema_version": "1.2",
  "canon_ref": "<sha from canon.lock.json>",
  "id": "ep01-title",
  "episode": 1,
  "title": { "ko": "제목", "en": "Title" },
  "timeline": "2025-01-15",
  "synopsis": { "ko": "요약", "en": "Summary" },
  "characters": ["character-id"],
  "locations": ["location-id"],
  "contributor": "github-username",
  "canon_status": "canonical",
  "themes": ["theme"],
  "canon_events": ["event-id"],
  "word_count": { "ko": 3200, "en": 2800 },
  "temporal_context": {
    "prev_episode": "ep00-slug",
    "next_episode": null,
    "thematic_echoes": ["ep01-related"]
  }
}
```

Required: `schema_version`, `canon_ref`, `id`, `episode`, `title`, `timeline`, `synopsis`, `characters`, `locations`, `contributor`, `canon_status`

## canon.lock.json

Generated by `canon lock`. Do not edit manually.

```json
{
  "schema_version": "canon.lock.v2",
  "canon_commit": "<git-HEAD-sha>",
  "worldbuilding_hash": "<sha256-of-canon-dir>",
  "hash_algo": "sha256",
  "generated_at": "2025-01-15T00:00:00.000Z",
  "contributors": ["github-username"]
}
```

`canon_ref` in metadata.json must match `canon_commit` here.

## 7 compliance checks

| ID | Rule |
|---|---|
| `metadata_schema_valid` | Required fields, `schema_version === "1.2"` |
| `characters_valid` | `characters[]` IDs exist in `canon/characters/` |
| `locations_valid` | `locations[]` IDs exist in `canon/worldbuilding/locations/` |
| `timeline_consistent` | `timeline` is `YYYY-MM-DD` |
| `continuity_valid` | `prev_episode`, `next_episode`, `thematic_echoes` resolve |
| `canon_version_match` | `canon_ref` matches `canon.lock.json` |
| `contributor_valid` | `contributor` is non-empty |
